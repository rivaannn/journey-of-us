# Journey 2025

Aplikasi web interaktif yang dibangun untuk merayakan perjalanan 5 tahun hubungan. Proyek ini menampilkan berbagai momen spesial melalui antarmuka yang modern dan responsif.

## Tech Stack
- **Framework:** React + Vite
- **Styling:** Tailwind CSS
- **Animation:** Framer Motion
- **Icons:** Lucide React
- **Language:** TypeScript

## Fitur Utama
- **Timeline:** Visualisasi perjalanan hubungan dari tahun ke tahun.
- **Galeri:** Kumpulan foto kenangan dengan mode slider dan grid.
- **Surat:** Koleksi pesan personal dengan tampilan amplop interaktif.
- **Musik:** Pemutar musik latar belakang yang persisten.
- **Game:** Permainan memori sederhana bertema pasangan.
